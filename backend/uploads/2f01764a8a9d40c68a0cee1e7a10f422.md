Выбор модели в этом выпадающем списке должен сохраняться в таблицу settings под нужным id подходящим пользователю (получение настроек пользователя /api/Settings/{userId} ), сохранение изменений по /api/Settings/save {
  "id": 0,
  "stream": true,
  "model": "string",
  "temperature": 0,
  "maxTokens": 0
}

Пусть при выборе модели из выпадающего списка изменения автоматически сохранялись, при "сбросе настроек" модель выбирается просто самая первая из списка, а при "Проверить" должен выполняться запрос /api/Ai/models, обновляться список моделей и выводиться статус подключения успешно или нет
