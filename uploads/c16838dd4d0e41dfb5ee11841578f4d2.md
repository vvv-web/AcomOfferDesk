— позволяют функции принимать бесконечное число аргументов. Не могут устанавливать лимиты.
```ts
function myltiplyBy(by: number, ...numbers: number[]) {
	return numbers.map((eachNumber) => by * eachNumber);
}
```
Для установки лимитов используются Turples.
```ts
const args2: [number, number] = [8, 5];
// Math.atan2(x, y)
const angle2 = Math.atan2(...args2);
```
