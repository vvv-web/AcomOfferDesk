Работает с разными типами и логикой, адаптируясь вводному типу.
### Polymorphism with Generics
Generics provide parametric polymorphism: работает с any и поддерживает безопасность типов.
```ts
function identity<T>(value: T): T {
	return value;
}
```
### Polymorphism with Inheritance
```ts 
abstract class Shape { abstract area(): number; }
class Circle extends Shape { /*..*/ }
class Rectangle extends Shape { /*..*/ }
```
### Polymorphic Function using Union Types
```ts
function print(val: string | number) {
	console.log(val);
}
```
### Polymorphism with Conditional Logic
```ts
function say(x: any) {
	if (typeof x === "number") { /* Number logic */ }
	else if (typeof x === "object") { /* Object logic */ }
	else { /* Default logic */ }
}
```
